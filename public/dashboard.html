About Taskify

Taskify is an open-source backend system that simplifies task management via a secure REST API. It's built with Node.js, Docker, and deployed on OpenShift.

Features

🧠 Smart Tasks

Manage, update, and delete tasks easily.

🔐 Authentication

Secure login with JWT tokens.

🚀 Deployable

Dockerized and ready for OpenShift.

📦 API Ready

Connect from any frontend app.
